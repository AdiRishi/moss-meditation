# Brand Deck

The brand deck is available in three forms:

- `zen-brand-deck.pptx` — editable 16-slide source deck.
- `zen-brand-deck.pdf` — portable review export.
- `slides/` — one PNG render per slide for visual inspection and implementation reference.

The deck connects Zen's product purpose to its identity, colour, typography, layout, components, motion, sound, architecture, complete screen system, accessibility rules, and implementation handoff. It is ready for review but is not yet an approved implementation specification.
