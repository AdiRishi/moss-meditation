# Zen Visual Generation Prompts

These prompts record the intent used to create the high-fidelity boards. Every generation used the prior Still deck as the selected visual source: warm editorial minimalism, mist-and-limestone surfaces, Ink typography, restrained Moss accents, delicate line icons, photographic quiet landscapes, generous negative space, and premium iOS presentation. The product name is **Zen**.

Shared constraints for every board:

- Preserve the selected visual system; do not introduce a competing art direction.
- Use `#F5F2EB`, `#E6DFD2`, `#CFC6B8`, `#7B866E`, `#4B5154`, and `#1E2326`.
- Combine refined serif display type with a highly legible modern sans-serif.
- Render realistic black iPhone frames on a warm off-white editorial board.
- Keep copy short, calm, and non-judgemental.
- Do not add guided meditation, content feeds, ambient audio, gamification, or account surfaces.

## 01 — Onboarding

Create one 16:9 high-fidelity product-design board showing five sequential Zen iPhone screens: splash with the ensō and tagline, welcome, flexible weekly practice goal, optional morning/evening schedule, and a gentle reminder-permission screen. Make each screen implementation-legible and visually continuous with the supplied prior brand deck.

## 02 — Daily practice

Create one 16:9 high-fidelity board showing three Zen iPhone screens: Today with the next planned practice and a Begin action; session duration setup with 5, 10, 15, 20, and 30 minute choices; and completion-sound selection with restrained preview controls. The app is silent during meditation and the selected sound plays only once at completion.

## 03 — Meditation flow

Create one 16:9 high-fidelity board showing three Zen iPhone screens: an active silent session with remaining time and one pause control; the final moments with “Gently returning”; and session complete with duration, time, optional feeling chips, and Done. Use a subtle breathing field, not instructional content or an audio player.

## 04 — Progress and settings

Create one 16:9 high-fidelity board showing five Zen iPhone screens: weekly rhythm overview, month practice history, schedule management, reminder settings with quiet hours, and settings overview. Progress should encourage return without punishing missed days or using celebratory gamification.

## 05 — Components and states

Create one 16:9 implementation-oriented board for Zen's complete UI system: primary and secondary actions, icon button, duration and weekday selectors, toggles, session and schedule cards, progress indicators, three-destination navigation, completion-sound row, bottom sheet, loading/empty/offline/error states, list items, line icons, and the exact colour palette. Keep the system quiet, spacious, accessible, and consistent with the generated screens.

## Provenance

The prompts synthesize the project owner's product direction in [`VISION.md`](../VISION.md) and [`PRODUCT.md`](../PRODUCT.md), the selected references in `previous-design-work/`, and the implementation rules in [`BRAND.md`](./BRAND.md) and [`SCREENS.md`](./SCREENS.md). No external visual source or unrelated brand system was used.
